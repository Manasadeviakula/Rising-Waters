# 🎥 FloodPredictor – Project Demo

This repository contains the demonstration video of **FloodPredictor**, an AI-powered Flood Risk Prediction System developed using Machine Learning and Flask.

## 📌 About the Project

FloodPredictor predicts the likelihood of flooding based on environmental and weather-related parameters. The application provides a simple and interactive interface for users to enter required data and receive instant predictions using a trained XGBoost Machine Learning model.

## 🚀 Demo Highlights

The video demonstrates:

- 🏠 Modern Home Page
- 🤖 AI-Based Flood Prediction
- 📝 User Input Form
- ⚡ Real-Time Prediction
- 📊 Prediction Results
- 🎨 Responsive User Interface
- 🌐 Flask Web Application

## 🛠 Tech Stack
